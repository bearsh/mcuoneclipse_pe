Component XFormat
Utililty with printf() like functionality
Component Level: High
Category: SW-Tools Library
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • xvformat - Printf() like format function

      ANSIC prototype: unsigned xvformat(void (*outchar)(void *,char), void *arg, const char * fmt, va_list args)
    • outchar:byte - Function pointer to the function to output one char.
    • arg:void* - Argument for the output function.
    • fmt:byte - Format options for the list of parameters.
    • args:byte - List of parameters
    • Return value:unsigned - Error code

  • xformat - Printf() like function using variable arguments

      ANSIC prototype: unsigned xformat(void (*myoutchar)(void *,char), void *arg, const char * fmt, ...)
    • myoutchar:byte - Function pointer to output one new character
    • arg:byte - Argument for the output function
    • fmt:byte - Format options for the list of parameters
    • openArgList:byte - Open argument list
    • Return value:unsigned - Error code

  • xsprintf - sprintf() like function

      ANSIC prototype: int xsprintf(char *buf, const char *fmt, ...)
    • buf: Pointer to char - Pointer to buffer to be written
    • fmt: Pointer to char - Pointer to formatting string
    • argList:byte - Open Argument List
    • Return value:int - number of characters written, negative for error case


Note: Some methods can be implemented as macros.
