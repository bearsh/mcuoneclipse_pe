Component S65Sharp
Display driver for the Siemens S65 display (using Sharp LS020xxx controller)
Component Level: Low
Category: CPU External Devices-Display
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • GetWidth - Returns the width of the display in pixels

      ANSIC prototype: ComponentName_PixelDim GetWidth(void)
    • Return value:ComponentName_PixelDim - Width of the display in pixels

  • GetHeight - Returns the height of the display in pixels

      ANSIC prototype: ComponentName_PixelDim GetHeight(void)
    • Return value:ComponentName_PixelDim - Height of the display in pixels

  • GetLongerSide - Returns the longer size of the display

      ANSIC prototype: ComponentName_PixelDim GetLongerSide(void)
    • Return value:ComponentName_PixelDim - Size of the longer side of the display

  • GetShorterSide - Returns the shorter size of the display

      ANSIC prototype: ComponentName_PixelDim GetShorterSide(void)
    • Return value:ComponentName_PixelDim - Size of the shorter side of the display

  • GetDisplayOrientation - returns the current orientation of the display

      ANSIC prototype: ComponentName_DisplayOrientation GetDisplayOrientation(void)
    • Return value:ComponentName_DisplayOrientation - current orientation of the display

  • SetDisplayOrientation - Sets the current orientation of the display

      ANSIC prototype: void SetDisplayOrientation(ComponentName_DisplayOrientation newOrientation)
    • newOrientation:ComponentName_DisplayOrientation - new orientation of the display

  • Clear - Clears the whole display memory.

      ANSIC prototype: void Clear(void)

  • UpdateFull - Updates the whole display. This is only a stub for this display as we are using windowing.

      ANSIC prototype: void UpdateFull(void)

  • UpdateRegion - Updates a region of the display. This is only a stub for this display as we are using windowing.

      ANSIC prototype: void UpdateRegion(ComponentName_PixelDim x, ComponentName_PixelDim y, ComponentName_PixelDim w, ComponentName_PixelDim h)
    • x:ComponentName_PixelDim - x coordinate
    • y:ComponentName_PixelDim - y coordinate
    • w:ComponentName_PixelDim - widht of the region
    • h:ComponentName_PixelDim - Height of the region

  • Hide - Using this command, the content of the display can be made hidden (either with a white or black screen). The content of the display will not be lost.

      ANSIC prototype: void Hide(bool black)
    • black:bool - if set to TRUE, then the display will be black, otherwise it will be white.

  • Unhide - Shows the content of the display again after a Hide() call.

      ANSIC prototype: void Unhide(void)

  • Shutdown - Used to shut down the display (to extend its life). Just switching the power off may reduce the life span of the display.

      ANSIC prototype: void Shutdown(void)


Note: Some methods can be implemented as macros.
