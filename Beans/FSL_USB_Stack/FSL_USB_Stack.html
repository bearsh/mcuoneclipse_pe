Bean FSL_USB_Stack
Freescale USB Stack
Component Level: High
Category: SW-Communication
This component implements a Processor Expert Wrapper to the Freescale USB Stack. To have this component working, you still need to initialize the USB peripheral using the the INIT_USB (or equivalent) components. See the examples on Embedded Components and MCU on Eclipse for details.
