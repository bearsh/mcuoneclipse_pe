Component App_SMAC_Hello
Simple SMAC 'Hello' Application
Component Level: High
Category: SW-User Components
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Sender - If the application is the sender (otherwise it is the receiver).
  • Hello String - String the sender shall send to the receiver
  • Acknowledge String - Acknowledge string the receiver shall send back to the sender
  • Delay (ms) - Delay used for communication. Lower delays will increase the number of packets sent.
  • HW Interface - Interface to the hardware
    The following items are available only if the group is enabled (the value is "Enabled"):
    • LED0 - LED0 is used to indicate good communication
      The following items are available only if the group is enabled (the value is "Enabled"):
      • LED 0 - Interface to LED0, used to indicate good communication.
    • LED1 - LED1 is used to indicate bad communication together with LED2
      The following items are available only if the group is enabled (the value is "Enabled"):
      • LED 1 - Interface to LED1, used to indicate bad communication (e.g. timeout).
    • LED2 - LED2 is used to indicate bad communication together with LED1
      The following items are available only if the group is enabled (the value is "Enabled"):
      • LED 2 - Interface to LED2
    • LED3 - LED3 is used only during initialization of the system
      The following items are available only if the group is enabled (the value is "Enabled"):
      • LED 3 - Interface to LED3
    • Communication - Settings for communication
      • Output Power - Specifies the output (sending) power. 0 means minimal power and 15 maximum power.
      • Channel - Channel to be used for communication
      • Clock Rate - Initial clock rate (CLKO, Clock Out) of the transceiver. This can be used as clock for the CPU.

        There are 8 options:
        • 16 MHz: Recommended default
        • 8 MHz:
        • 4 MHz:
        • 2 MHz:
        • 1 MHz:
        • 6.25 kHz:
        • 32.786 kHz:
        • 16.393 kHz:

      • RX Sender Timeout - Timeout value until the sender expects the Acknowledge from the receiver (0 disables the timeout).
      • RX Receiver Timeout - Timeout value until the receiver expects a message from the sender (0 disables the timeout, means the receiver is waiting/blocking until it receives a packet from the sender).
      • Max Data Length - The maximum packet size we are interested in. Useful to filter out any packets larger than this size.
      • Communication Interface - Interface to the SMAC wireless stack (802.15.4)
    • Terminal Interface - If enabled, it uses a terminal to write messages
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Terminal - Writes messages to the terminal
    • LCD Interface - Interface to the LCD display
      The following items are available only if the group is enabled (the value is "Enabled"):
      • LCD - Interface to the LCD display
  • System Interface - Interface to the system environment
    The following items are available only if the group is enabled (the value is "Enabled"):
    • Wait Interface - Interface to waiting infrastructure
    • Utility Interface - Utility module with helper functions.
    • Events Interface - Interface to the events module
