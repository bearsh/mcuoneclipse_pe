Component USB_Host_CMX
USB Host driver (based on CMX)
Component Level: Low
Category: SW-Communication
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • CPU Derivative - CPU to be used

    There are 2 options:
    • MCF51JM128: MCF51JM128
    • MCF52259: MCF52259

  • Keyboard - Group defining the keyboard
    • SHIFT key - If the SHIFT key shall be supported
    • ALT key - If the ALT key shall be supported
    • CTRL key - If the CTRL key shall be supported
    • GUI key - If the GUI key shall be supported
  • BDT Base - Buffer Descriptor Base: Buffer in RAM, aligned at 512 byte boundaries, and 512 (0x200) bytes large. Make sure you have reserved this space in your linker file.
  • USB Internal Pullups - This enables the on chip internal USB pull-up. Otherwise an external pull-up is required
  • USB Internal Voltage Regulator - This enables the internal voltage regulator (3.3V) which provides a stable power source to power the USB internal transceiver. If enabled, you have to provide 3.9-5.5V Vdd, with no no external supply connected to VUSB33 pin. With disabled voltage regulator, the MCU and the USB can operate at different voltages, and a 3.3V source must be provided through the Vusb33 pin to power the USB transceiver.
  • Wait - Interface to wait module
