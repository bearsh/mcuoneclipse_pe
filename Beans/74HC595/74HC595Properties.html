Component 74HC595
Driver for the 74HC595 Shift Register Device
Component Level: High
Category: SW-User Components
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Latch (STCP) - STCP, Storage register clock input. A pulse on this pin latches the data to the output pins
  • Data (DS) - DS, Serial Data Input. This pin provides the serial data.
  • Clock (SHCP) - SHCP, Shift register clock input. This is the clock pin to the device.
  • Clock/Latch delay (ns) - Latch/Clock delay in nano-seconds, zero for no delay. The clock and latch signal might need some additional delays. The datasheet specifies 16-19 ns.
  • Wait - Interface to waiting routines
  • MSB first - If the Most Significant Bit of a byte shall be shifted first. Otherwise shifting starts with the LSB (Least Significant Bit).
  • Enable Pin (OE) - If enabled, the driver is using the OE (Enable pin). Otherwise, the pin needs to be to a high level to have the device enabled)
    The following items are available only if the group is enabled (the value is "Enabled"):
    • Enable in Init - If device is enabled (OE low) during startup and Init() call.
    • Enable (OE) - OE, Output Enable, low active. If high, the device output pins are in Z-state (OFF)
