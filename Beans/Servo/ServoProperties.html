Component Servo
Driver for a Servo
Component Level: High
Category: SW-Motor Control
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • Inverted PWM - If the PWM signal shall be inverted.
  • Init PWM (us) - Initialization duty cycle for PWM in microseconds.
  • Min Pos PWM (us) - Minimal Servo PWM value, in microseconds. This is used for the position calculation and corresponds to position 0.
  • Max Pos PWM (us) - Maximal Servo PWM value, in microseconds. This is used for the position calculation and corresponds to position 255.
  • PWM Period (us) - PWM Period in us. Typically 20'000 us (50 Hz) are used. This needs to match the PWM settings used.
  • System - Group of interfaces to system
    • Timed Move - If enabled, it is possible to have a timed move of the servo.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Trigger - Interface to the trigger module
    • Shell - If enabled, the component has a command line shell interface implemented.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Shell - Interface to shell
      • Utility - Interface to utility module
    • PWM - Servo PWM signal
  • CPU clock/speed selection - Settings for the CPU speed modes: if the component is supported or not.
    • High speed mode - The component is enabled/disabled in high speed mode.
