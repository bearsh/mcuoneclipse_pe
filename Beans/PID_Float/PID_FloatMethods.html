Component PID_Float
PID implementation using floating variable.
Component Level: High
Category: SW-Control
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • Control - Control function. Must be called every T time.

      ANSIC prototype: void Control(float error, void* *u)
    • error:float - Error signal. (Reference - SystemOutput).
    • u: Pointer to void* - Pointer to the control variable.

  • Reset - Reset the PID.

      ANSIC prototype: void Reset(void)

  • Set_K - Change the K parameter.

      ANSIC prototype: void Set_K(float k)
    • k:float - New K value.

  • Set_Ti - Change the Ti parameter.

      ANSIC prototype: void Set_Ti(float t)
    • t:float - New Ti value.

  • Set_Td - Change the Td parameter.

      ANSIC prototype: void Set_Td(float t)
    • t:float - New Td value.

  • Set_T - Change the T parameter.

      ANSIC prototype: void Set_T(float t)
    • t:float - New T value.


Note: Some methods can be implemented as macros.
