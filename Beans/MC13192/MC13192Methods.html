Component MC13192
Device Driver for the Freescale MC13192 Transceiver
Component Level: Low
Category: CPU External Devices-Communication
Methods:
(Methods are user-callable functions/subroutines intended for the component functions control. Please see the Embedded Components page for more information.)
  • EnableInterrupts - Enables transceiver interrupts

      ANSIC prototype: void EnableInterrupts(void)

  • DisableInterrupts - Disables Transceiver interrupts.

      ANSIC prototype: void DisableInterrupts(void)

  • RTXENDeAssert - no hint

      ANSIC prototype: void RTXENDeAssert(void)

  • RTXENAssert - Asserts the RTXEN signal.

      ANSIC prototype: void RTXENAssert(void)

  • Wake - Wake up the transceiver.

      ANSIC prototype: void Wake(void)

  • Write - Writes a command to the transceiver

      ANSIC prototype: void Write(byte addr, word val)
    • addr:byte - Transceiver address
    • val:word - Value to write to the register

  • Write4 - Writes 32bit (dword) data to the transceiver. The data is in big endian format, and it writes it to the transceiver (which has little endian memory layout).

      ANSIC prototype: void Write4(byte addr, dword val)
    • addr:byte - Transceiver start address where to write the data.
    • val:dword - Value to write to the register, in big endian format.

  • Read - Read a word from the transceiver

      ANSIC prototype: word Read(byte addr)
    • addr:byte - Address in the transceiver to read from.
    • Return value:word - Error code

  • Read2 - Read the second word from the transceiver.

      ANSIC prototype: word Read2(byte addr)
    • addr:byte - Address in the transceiver to read from.
    • Return value:word - Error code

  • WriteTx - Writes a block of data to the TX packet RAM of the transceiver.

      ANSIC prototype: void WriteTx(tTxPacket *psTxPkt)
    • psTxPkt: Pointer to byte - Pointer to

  • ReadRx - Writes a block of data to the TX packet RAM of the transceiver.

      ANSIC prototype: byte ReadRx(tRxPacket *psRxPkt)
    • psRxPkt: Pointer to byte - Pointer to
    • Return value:byte - status

  • Restart - Restarts the transceiver

      ANSIC prototype: void Restart(void)

  • ContReset - Reset (continuous) the transceiver

      ANSIC prototype: void ContReset(void)

  • SetClockRate - Set the CLKO frequency

      ANSIC prototype: byte SetClockRate(byte u8Freq)
    • u8Freq:byte - frequency to be used. 0 = 16 MHz, 1 = 8 MHz, 2 = 4 MHz, 3 = 2 MHz, 4 = 1 MHz, 5 = 62.5 kHz, 6 = 32.768+ kHz, 7 = 16.393+ kHz
    • Return value:byte - Error code

  • RadioInit - no hint

      ANSIC prototype: void RadioInit(void)

  • InitCommChannel - Method to init or re-initialize the communication channel. This method is used if the SPI communication channel is shared among different peripherals.

      ANSIC prototype: void InitCommChannel(void)

  • GetRadioSPI - Method called in case of a shared bus used to the transceiver.

      ANSIC prototype: void GetRadioSPI(void)

  • LeaveRadioSPI - Method called in case of a shared bus used to the transceiver.

      ANSIC prototype: void LeaveRadioSPI(void)

  • CheckRx - Needs to be called after interrupt received from the transceiver to check the state and to create the events.

      ANSIC prototype: byte CheckRx(tRxPacket *sRxPacket)
    • sRxPacket:byte - no hint
    • Return value:byte - Error code


Note: Some methods can be implemented as macros.
