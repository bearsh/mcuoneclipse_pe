Driver for the EGBT-045MS e-Gizmo Bluetooth Module
Component Level: High
Category: SW-User Components

HC06 specific information.
The HC06 has a very limited firmware compared to the HC05. Unlike the HC05, the HC06 is *not* having \r\n at the end of the commands. Default settings of the HC-06:
Slave
9600, N, 8, 1
Name: linvor
Pincode: 1234

Sent: AT
Receive: OK


Sent: AT+VERSION
Receive: Linvor1.6

Setting name to 'blue':
AT+NAMEblue
response:OKsetname

Change baud rate

Sent: AT+BAUD#
Receive: OKbaud

1: 1200
2: 2400
3: 4800
4: 9600
5: 19200
6: 38400
7: 57600
8: 115200

Example
Sent: AT+BAUD1
Receive: OK1200

Change pin code

Sent: AT+PINxxxx
Receive: OKsetpin

Example
Sent: AT+PIN9999
Receive: OKsetpin
