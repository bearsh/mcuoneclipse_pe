Component MMA7260Q
Driver for the Freescale 3-Axis Accelerometer MMA7260Q
Component Level: Low
Category: CPU External Devices-Sensor
Properties:
(Properties are parameters of the component. Please see the Embedded Components page for more information.)
  • Component name - Name of the component.
  • HW Interface - Provides Settings to the hardware interface
    • G Select - The device has internal pull downs, so if the g select pins are not connected, the device operates in 1.5g mode.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • G Select 1 Signal - G Select Pin 1
      • G Select 2 Signal - G Select Pin 2
    • Sleep - The sleep signal is low active. So if you do not connect this pin, then you need to have it set to a logical one level. Sleep mode is controlled using SensorOn() and SensorOff() methods.
      The following items are available only if the group is enabled (the value is "Enabled"):
      • Sleep Signal - The sleep signal is low active and allows to put the sensor into a sleep mode reducing energy consumption.
      • Enabled in init - If the sensor shall be enabled in the initialization code. Disabling the sensor will reduce power consumption.
      • Enable/Disable A/D - If the A/D Converter shall be enabled and disabled during SensorOn() and SensorOff() to reduce power consumption.
    • A/D -
      • A/D Converter - A/D converter to read in the acceleration analog values.
      • X channel - Channel on A/D converter for the X value.
      • Y channel - Channel on A/D converter for the Y value.
      • Z channel - Channel on A/D converter for the Z value.
  • Calibration - Contains initialization calibration values.
    • X offset - Offset to be added to X acceleration sensor value
    • X 1g value - Value of the X axis for +1g
    • Y offset - Offset to be added to Y acceleration sensor value
    • Y 1g value - Value of the Y axis for +1g
    • Z offset - Offset to be added to Z acceleration sensor value
    • Z 1g value - Value of the Z axis for +1g
  • System - Defines the overall system configuration.
    • Wait - Interface to busy wait.
